# Personal Portfolio Website

A modern and responsive personal portfolio website built using HTML, Tailwind CSS, and JavaScript.

## Features
- Responsive Design
- Dark/Light Mode
- About Me Section
- Skills Showcase
- Projects Section
- Achievements & Certifications
- Contact Form
- Social Media Links

## Tech Stack
- HTML
- Tailwind CSS
- JavaScript

## Run
1. Download the project.
2. Open `index.html` in your browser.
